# **App Name**: ApniSec

## Core Features:

- Database Connection: Establish a connection to a MongoDB database using Mongoose with singleton pattern to avoid connection exhaustion.
- API Response: Base class providing a consistent success response with data and message.
- API Error: Custom error class that standardizes error responses from the API and allows specific HTTP codes

## Style Guidelines:

- Primary color: Teal (#008080) to create a feeling of modern technical prowess, referencing designs on ApniSec webpage.
- Background color: Very dark gray (#121214), same hue as primary with low saturation.
- Accent color: Green (#00A300) for a highlight tone
- Font pairing: 'Space Grotesk' (sans-serif) for headers, and 'Inter' (sans-serif) for body text.
- Code font: 'Source Code Pro' (monospace) for displaying code snippets.
- Simple line icons for visual appeal, consistent with the aesthetic of the ApniSec design
